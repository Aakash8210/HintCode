# HintCode 🧠

> "Don't just solve problems. Learn to think through them."

HintCode is a DSA learning platform built for CSE students who are 
tired of getting full solutions from ChatGPT when all they needed 
was a nudge in the right direction.

## The Problem
Every CS student practices on LeetCode. But the moment they get 
stuck, they ask AI for help — and AI dumps the entire solution. 
That kills the learning.

## The Solution
HintCode connects to LeetCode, fetches real problems, and guides 
you with **progressive Socratic hints** — from conceptual nudges 
to pseudocode — never giving you the answer until you ask for it.

## Features
- 🔍 Fetch any LeetCode problem by number or title
- 💡 Progressive hint system (5 levels — conceptual → code)
- 🧩 AI-powered problem breakdown in plain English
- 🐛 Debug feedback that points you in the right direction
- ▶️ Run code against example + edge case test cases
- 📖 Full solutions with multiple approaches (when you're ready)
- 🌐 Python3, Java, C++ support with official LeetCode snippets

## Built With
Next.js · Monaco Editor · Gemini API · JDoodle API · LeetCode GraphQL
